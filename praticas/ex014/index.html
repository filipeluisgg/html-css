Capítulo 1.1 - CSS local, Como funciona ↓ ↓ ↓ ↓

Atualmente, há 3 técnicas de CSS, a primeira foi explicada no último arquivo que eu fiz, a segunda que está sendo usada nesse site, é a técnica de CSS local/interno, usada em documentos de poucas páginas, umas 3 por exemplo.

Essa técnica, sendo mais vantajosa que a primeira por não poluir o html e precisar ser usada apenas uma vez, ainda conta com algumas desvantagens, tais como ser feita no mesmo documento/página que o HTML, e geralmente ocupar mais linhas do que o própio HTML.
