Pretendências de carreira

Eu, Luis Felipe(criador deste site), estou aprendendo agora, HTML e CSS, após isso, pretendo conhecer o JavaScript, e com a combinação dessas três técnologias, poderei criar qualquer tipo de site que me convém, além de poder hospeda-los em algum servidor na internet para mecanismos de busca.

O que são essas técnologias?

A base de todo site é o HTML, CSS e o JavaScript. O HTML, abreviação de HiperText Markup Language, é uma linguagem de marcação, para "colocar conteúdo" nos sites, já o CSS, que significa Cascading Style Sheets, é uma linguagem de estilização, com a finalidade de organizar o conteúdo do HTML, deixando o site mais atraente.

As duas linguagens explicadas anteriormente não são linguagens de programação, diferentemente do JavaScript, ou JS como costumam abreviar. Essa linguagem poderosa, serve para fazer as interações necessárias para por exemplo, definir o que acontecerá quando se clica em um botão ou link que está em um site ou software.


O princípio da minha escolha

Primeiramente, por que eu escolhi ser um desenvolvedor? Apenas por gostar! em 2021, estudando educação financeira, foi vendo um podcast de uma pessoa que ganhou a vida, respondendo resumidamente a seguinte pergunta: "O que você faria, se voltasse para o começo?", eu, ouvindo a resposta da mesma, falando que se aprofundaria em python, e usaria essa linguagem de programação para ganhar uma ótima renda auferida, para a longo prazo transformar em renda passíva, eu decidi pesquisar sobre o que era python, foi assim que eu conheci Gustavo Guanabara, um dos melhores professores do Brasil, ensinando sobre python, eu adorei programar. E hoje em 2024, estou indo para o segundo ano de faculdade, cursando Análise e desenvolvimento de sistemas, e escrevendo meu princípio vocacional em um site ao qual não paro de aprender.
