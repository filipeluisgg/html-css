Capítulo 1.1 - CSS inline e sua utilização ↓ ↓ ↓ ↓

Atualmente, há 3 técnicas de CSS, a primeira é a CSS inline, que está sendo utilizada para fazer este site por completo. Esta técnica não é nada recomendada para ser utilizada na criação de um site por inteiro, devendo ser usada apenas para casos específicos e correções pequenas. Caso contrário, terá várias desvantagens, tais como ter de modificar linha por linha do código, desorganiza-lo, e poluir o HTML.
